# StudyLock 📚

A real-time collaborative study room with pixel-art library visuals, ambience sounds, and AI-powered adaptive environment.

**[studylock.dev](https://studylock.dev)**

## Features

- 🏛 **14 Unique Themed Floors** — Classic Library, Modern Study, Cozy Corner, Dark Room, and more
- 👥 **Real-time Presence** — See other students studying alongside you
- 🎵 **Ambient Sound System** — Lofi, Rain, Ocean, Birds, Fire, Cabin with mini player & visualizer
- ⏱ **Pomodoro Timer** — Focus/Break cycles with automatic room switching
- ☕ **Break Rooms** — Cafe Lounge & Outdoor Garden for break time
- 🌙 **Dark Room** — Spotlight effect for deep focus sessions
- 🔄 **Auto-expanding Floors** — New floors open as capacity fills

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js, TypeScript, Canvas 2D, Howler.js, Tailwind CSS |
| Backend | FastAPI (Python) |
| AI Agents | CrewAI, AutoGen, LangChain, LangGraph, MCP |
| Database | Supabase (PostgreSQL), Redis (Upstash) |
| Testing | Playwright (E2E) |
| CI/CD | GitHub Actions |
| Deployment | Vercel + Railway |

## Getting Started

```bash
cd frontend
npm install
npm run dev
```
