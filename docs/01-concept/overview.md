# StudyLock — Concept Overview

## One-Liner

A pixel-art library where people study together in real-time with adaptive ambience sounds.

## Why Build This

- "Study with me" YouTube videos have hundreds of millions of views → demand is proven
- People don't want to study alone
- No one has done this properly on the web

## Target Users

- Students (exams, assignments, finals season)
- Job seekers (coding tests, certifications)
- Remote workers (need focus environment)
- Freelancers

## Core Value Proposition

1. **You're not alone** — see other people studying when you join
2. **Immersive environment** — pixel-art visuals + ambience sounds
3. **Zero friction** — no login required, just join instantly

## Competitors

| Service | How StudyLock Differs |
|---------|---------------------|
| Studystream | Webcam-based — showing your face is uncomfortable |
| FocusMate | 1:1 matching — feels forced |
| Lifeat.io | Video backgrounds — not real-time |
| Lofi Girl | YouTube — no interaction |
| **StudyLock** | Pixel-art + anonymous + real-time + AI-adaptive environment |

## Viral Strategy

- Pixel-art library screenshot → "what is this, it's so cute" sharing
- "243 people studying right now" → sense of belonging
- Target subreddits: r/studytips, r/GetStudying, r/college
