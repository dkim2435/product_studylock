# Library Reference

## Frontend

| Library | Install | Docs |
|---------|---------|------|
| Next.js | `npx create-next-app@latest` | https://nextjs.org/docs |
| Howler.js | `npm install howler` | https://github.com/goldfire/howler.js |
| Tailwind CSS | `npm install -D tailwindcss postcss autoprefixer` | https://tailwindcss.com/docs |
| Supabase JS | `npm install @supabase/supabase-js` | https://supabase.com/docs |
| Canvas 2D | Built-in browser API | https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API |

## Backend

| Library | Install | Docs |
|---------|---------|------|
| FastAPI | `pip install fastapi uvicorn` | https://fastapi.tiangolo.com |
| Supabase Python | `pip install supabase` | https://supabase.com/docs |
| Redis | `pip install redis` | https://upstash.com/docs/redis |
