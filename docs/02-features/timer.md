# Pomodoro Timer

## Overview

25min focus / 5min break cycles. Simple.

## Features

- 25min focus → 5min break → repeat
- After 4 cycles → 15min long break
- Start / Pause / Reset buttons
- Notification sound (bell)

## UI

```
⏱ 23:45
[Start] [Reset]

Today: 🍅🍅🍅🍅 4 pomodoros completed
```

## Tech

- Frontend JavaScript timer
- Notification API for browser alerts
- Optional: save pomodoro history to Supabase (if logged in)
